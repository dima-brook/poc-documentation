Xmessage Module

Overview

WIP

The xmessage module provides cross-chain message synchro functionality, by cross-consensus messages(XCM) & XP Protocol.

The xmessage module provides a PoC function for

  • Broadcasting an XP Protocol Message

Functionality

The Poc Function takes a Vec<u8> message and a MultiLocation(external Pallet) Destination. The message is encoded into XPProtocol format and sent to the target chain with XCMP protocol.

There must be an HRMP channel open between the two chains.
